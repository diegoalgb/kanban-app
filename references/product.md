# Product: Kanban Board Demo

## Overview

A single-page Kanban board application built to demonstrate simple app creation with Next.js and React. No backend, no database, no authentication — just a clean, interactive UI for managing tasks.

## Core Features

- **Kanban board** with columns (e.g. To Do, In Progress, Done)
- **Create cards** — add new task cards to any column
- **Move cards** — drag cards between columns to update their status
- **Delete cards** — remove cards that are no longer needed

## Scope

This is a demo application. All state is ephemeral (in-memory or local), and there is no user login, persistence layer, or API integration. The purpose is to showcase a functional, well-structured single-page app using the chosen tech stack.

## Tech Stack

- Next.js (App Router)
- React + shadcn/ui components
