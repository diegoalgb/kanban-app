# Kanban App

A simple Kanban web application for task management.

## Tech Stack

- **Framework**: Next.js (App Router)
- **UI**: React + shadcn/ui
- **Runtime**: Node.js

## Getting Started

```bash
npm install
npm run dev
```

## Project Goals

Build a simple, clean Kanban board where users can create, move, and manage tasks across columns (e.g. To Do, In Progress, Done).
